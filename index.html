FAQs

What is Frontend Mentor, and how will it help me?

Frontend Mentor offers realistic coding challenges to help developers improve thier frontend coding skills with projects in HTML, CSS and JavaScript. It's suitable for all levels and ideal for portfolio building.

Is Frontend Mentor free?

Frontend Mentor provides both free and premium options for its users. Frontend Mentor is a platform that offers coding challenges and projects to help web developers improve their skills.

Can I use Frontend Mentor projects in my portfolio?

Frontend Mentor encourages users to include the projects they complete on the platform in their portfolios. The projects provided by Frontend Mentor are designed to be used as a learning resource, and showcasing them in your portfolio is a great way to demonstrate your skills to potential employers or clients.

How can I get help if I'm stuck on a challenge?

If you're stuck on a challenge, there are several steps you can take to seek help and overcome obstacles like Review the Documentation, Search for Solutions, Check Community Forums , Ask for Help Online, Use Social Media or React out to Mentors or Peers.
